# Data Collection Screen Mockups

These SVG mockups were created for the manual occupancy + live microphone capture screen.

Files:

- `data-collection-screen-mockup.svg`: full screen concept
- `element-decibel-readout.svg`: numeric decibel treatment
- `element-noise-bar.svg`: qualitative noise gradient bar
- `element-occupancy-control.svg`: manual occupancy adjustment card
- `element-mic-wave-system.svg`: stable mic with background emanation concept

The color scale matches the existing heatmap language already used in `Web_Frontend/src/components/MapExplorer.tsx`.
